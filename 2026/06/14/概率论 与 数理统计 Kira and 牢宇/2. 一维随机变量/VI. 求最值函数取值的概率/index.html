VI. 求最值函数取值的概率

（1）最大值函数

对最大值，有

{a < max (X, Y) < b} = {max (X, Y) < b} − {max (X, Y) ≤ a}.

连续情形下常写为

P{a < max (X, Y) < b} = P{X < b, Y < b} − P{X ≤ a, Y ≤ a}.

（2）最小值函数

对最小值，有

{a < min (X, Y) < b} = {min (X, Y) > a} − {min (X, Y) ≥ b}.

连续情形下常写为

P{a < min (X, Y) < b} = P{X > a, Y > a} − P{X ≥ b, Y ≥ b}.
