Cov(X, Y) 为协方差, 下文会介绍 由此可得 D(X ± Y) = DX + DY ⇔ X 与 Y 不相关. 特别地，若 X 与 Y 相互独立，则 D(X ± Y) = DX + DY. （3）乘积的方差 若 X 与 Y 相互独立，则 D(XY) = DX ⋅ DY + (EX)2DY + (EY)2DX. 线性组合的方差展开时，常数项忽略；组合中有相关项时应写协方差，不相关时协方差为 0，独立一定不相关。 [!note] 常数的方差为0.
